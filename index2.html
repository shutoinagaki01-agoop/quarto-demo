  1. まずは、whisperを試してみました。
    whisperは、OpenAIが開発した汎用的な音声認識アプリです。詳細はこちら
    国会答弁の音声約30秒の文字起こしを行いました。(音声ファイルはpaperを参照)
    パラメータ数で5つのモデルが提供されており、今回はbaseとlargeで検証を行いました

import whisper

  1. 次に、Stable Diffusionを試してみました。
    stable diffusionは、テキストから画像を出力する拡散モデルです。詳細はこちら
Code 
%%time
from PIL import Image
from diffusers import DiffusionPipeline
 
model_id = "CompVis/ldm-text2im-large-256"
 
# load model and scheduler
ldm = DiffusionPipeline.from_pretrained(model_id)
 
# run pipeline in inference (sample random noise and denoise)
prompt = "A cat sleeping on cloud"
images = ldm([prompt], num_inference_steps=10, eta=0.3, guidance_scale=6)["sample"]
 
# save images
for idx, image in enumerate(images):
    image.save(f"sd-{idx}.png")

#show
im = Image.open('sd-0.png')
im

Wall time: 33.4 s

  1. ipyleaflet
    ipyleafletは、地理空間データを地図上に可視化するライブラリです。
    類似するものにfoliumがありますが、ipyleafletは、時系列と共に変化する地理空間データを動的に描画することが可能です。
Code 
%%time
from ipyleaflet import Map, Marker, basemaps, basemap_to_tiles
m = Map(
  basemap=basemap_to_tiles(
    basemaps.NASAGIBS.ModisTerraTrueColorCR, "2017-04-08"
  ),
  center=(52.204793, 360.121558),
  zoom=4
)
m.add_layer(Marker(location=(52.204793, 360.121558)))
m.add_layer(Marker(location=(22.204793, 300.121558)))
